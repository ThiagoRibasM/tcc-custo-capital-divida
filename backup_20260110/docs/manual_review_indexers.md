## 3. Sugestões de Tratamento


### 3.1 Padrões Comuns nos Não Identificados


1. **Percentuais diretos sem indexador**: Tratar como pré-fixado
2. **Variação cambial**: Adicionar tratamento específico
3. **Moedas estrangeiras**: Identificar e tratar separadamente
4. **Múltiplos indexadores**: Extrair o primeiro ou mais comum
5. **Formatos alternativos**: Expandir regex para capturar variações

### 3.2 Casos Extremos


1. **Kd muito alto**: Verificar conversão de período mensal para anual
2. **Kd muito baixo**: Validar se spread negativo foi aplicado corretamente
3. **TR com Kd baixo**: Normal, TR tem valor base muito baixo
4. **Pré-fixados internacionais**: Taxas podem ser corretas mas baixas
